# How to use the xPack Project

This section is intended for those who plan to use this
module in their
own projects.


The `web-preview-docusaurus` module can be imported into both TypeScript
and JavaScript Node.js code.

In TypeScript and ECMAScript modules, use `import`:

```typescript
import { Xxx } from 'web-preview-docusaurus'
```

In JavaScript with CommonJS, use `require()`:

```javascript
const { Xxx } = request('web-preview-docusaurus')
```
